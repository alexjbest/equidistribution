We will show

\begin{equation*} \frac{1}{\# \Omega_n} \sum_{x\in \Omega_n} P(x) \to 0 \end{equation*}

for \(\deg P(x) \gt 0\text{.}\) i.e.

\begin{equation} \sum_{\alpha \in \ZZ^3,|\alpha|^2 = n} P\left(\frac\alpha{|\alpha|}\right) = o( r_3(n))\tag{1.3} \end{equation}

where

\begin{equation*} r_3(n) = \# \{ a^2 + b^2 + c^2 = n: a,b,c\in \ZZ\}\text{.} \end{equation*}
