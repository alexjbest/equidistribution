If we were just looking for a modular form we might try Eisenstein series

\begin{equation*} E_k(z) = \sum_{c,d} \frac{1}{(cz+d)^{2k}} \end{equation*}

these are not cuspidal. But if we twist this a little bit

\begin{equation*} P_m(z; k) = \sum_{\gamma \in \Gamma_\infty \backslash \Gamma} j(\gamma; z)^{-2k} e(m \gamma z)\text{.} \end{equation*}
