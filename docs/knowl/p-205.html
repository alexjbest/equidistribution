\(y = \Im(z), y(\gamma z) = |cz+d| ^{-2} y\) so

\begin{equation*} F(z) = y^{k/2} |f(z)| \end{equation*}

Invariant on \(\Gamma\) bounded at the cusps \(|F(z)| \lt M\text{.}\) So

\begin{equation*} \int_0^1 f(z) e(nx) = e^{-2 \pi i ny }a_f(n) \end{equation*}
\begin{equation*} \le \int_0^1 y^{-k/2} M = O(y^{-k/2}) \end{equation*}

so \(a_f(n) = O(y^{-k/2} e^{2\pi i n y})\) with \(y = 1/n\text{.}\)
