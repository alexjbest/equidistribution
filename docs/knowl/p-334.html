  1. \(E(z,s)\) converges for \(\Re(s) \gg1\text{.}\)
  2. Has analytic continuation to \(\CC\) in the \(s\)-variable.
  3. Has a simple pole at \(s =1\) which is a constant \(\Res_{s=1} E(z,s) = \frac 12\text{.}\)
  4. \(E(z,s) = E(z,1-s)\text{.}\)
  5. \(E(z,s) \sim y^\sigma\) where \(\sigma = \max \{ \Re(s), \Re(1-s)\}\text{.}\)
