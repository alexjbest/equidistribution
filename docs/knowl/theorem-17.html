Theorem 1.52 Birkhoff ergodic theorem

\((X, B, \mu, T)\) a measure preserving system (as in the definition of ergodic) with \(\mu\) a probability measure. Then for any \(f \in L^1(X, \mu)\)

\begin{equation*} \lim_{n\to \infty} \frac 1n \sum_{i=0}^{n-1} f\circ T^i(x) = \bar f (x) \end{equation*}

for some \(\bar f \in L^1( X,\mu)\) moreover

\begin{equation*} \int \bar f \diff \mu = \int f \diff \mu \end{equation*}

and if \(T\) is ergodic then \(\bar f (x) = \int f \diff \mu\) almost everywhere.
