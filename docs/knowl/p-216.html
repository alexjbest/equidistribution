Using this we find

\begin{equation*} r_3(n^2) = 6 \sum_{d|N} w(d) S \left(\frac Nd\right) \end{equation*}

where for odd \(n\)

\begin{equation*} S(n) = \sum_{a+b = 2n,\,a,b\,\text{odd}} \rho(a) \rho(b) \end{equation*}

but

\begin{equation*} S(n) = \frac{1}{16}\#(2n = \underbrace{w^2 + x^2}_{\text{odd}}+ \underbrace{y^2 + z^2}_{\text{odd}}) \end{equation*}
\begin{equation*} = \frac{1}{4}\#(2n = 4w^2 + 4x^2+y^2 + z^2) \end{equation*}
\begin{equation*} = \sigma(n) \end{equation*}

by Jacobi.
