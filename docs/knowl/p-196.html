\(A = 4\times 4\) integral positive definite

\begin{equation*} Q = Q_A(x),\, x^\transpose A x \end{equation*}
\begin{equation*} r_Q(n) = \# \{ x \in \ZZ^4:Q(x) = n\} \end{equation*}
\begin{equation*} \theta_Q(z) = \sum_{n=0}^\infty r_Q(n) e(nz) \in M_2(\Gamma_0(N)) \end{equation*}

for some \(N = 4\det(A)\text{.}\)
