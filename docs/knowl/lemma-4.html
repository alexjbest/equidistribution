Lemma 1.43 Van der Corput

Let \(y_n \in \CC\text{,}\) such that \(y_n = 0\) if \(n \lt 1\) or \(n \gt N\) then

\begin{equation*} \forall H \in \ZZ_{\gt 0 } \end{equation*}

we have

\begin{equation*} \left| \sum_{n=1}^N y_n \right|^2 \le \frac{N+ H} {H+1 } \sum_{n=1}^N |y_n|^2 + \frac{2(N+H)}{H+1} \sum_{h=1}^H \left(1 - \frac h{H+1} \right) \left| \sum_{n=1}^{N-h} y_{n+h} \overline y_n\right|\text{.} \end{equation*}
