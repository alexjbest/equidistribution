Theorem 1.83 Petersson trace formula

For any \(m,n \ge 1 \)

\begin{equation*} \frac{\Gamma(k-1)}{ ( 4\pi \sqrt{mn})^{k-1}} \sum_{f\text{ o.n. basis for }S_k(\Gamma)} \overline a_f(m) a_f(n) = \delta_{m,n} + \frac{2\pi}{i^k} \sum_{c\equiv 0 \pmod N,\,c \gt 0} J_{k-1} \left( \frac {4\pi \sqrt{mn}}{c} \frac{K(m,n; c)}{c} \right)\text{.} \end{equation*}
