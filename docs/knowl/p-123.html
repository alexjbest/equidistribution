Let \(y_n \in \CC\text{,}\) such that \(y_n = 0\) if \(n \lt 1\) or \(n \gt N\) then we would like to bound

\begin{equation*} \sum_m \sum_n y_n \overline y_m = \sum_{m,n=1}^N = S_N \overline S_N = |S_N|^2\text{,} \end{equation*}

Weyl differencing gives

\begin{equation*} \sum_{m=1}^N \sum_{n=1}^N y_n \overline y_m = \underbrace{\sum_n |y_n| ^2}_{\text{diagonal contribution}} + \underbrace{2 \Re \left( \sum_{n=1}^{N-1} \sum_{m=1}^{N-n} y_{n+m} \overline y_m \right) }_{\text{off diagonal terms}}\text{.} \end{equation*}
