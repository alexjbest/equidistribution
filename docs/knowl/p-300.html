If \(d \ne q\) then \(de = q\text{,}\) \(e \ne 1\text{,}\) \(\alpha = \alpha_1 + d\alpha_2\) so

\begin{equation*} \sum_\alpha = \sum_{\alpha_1,d,\overline 4 \alpha_1 ^2 = 1 \pmod d} e\left( \frac{\alpha_1 u}{q}\right) \sum_{\alpha_2} e\left( \frac{\alpha_2 u}{q}\right) = 0\text{.} \end{equation*}
