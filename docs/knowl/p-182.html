Classically

\begin{equation*} \widetilde \theta(z) = \sum_{m\in \ZZ} e^{i \pi m^2z} \end{equation*}

converges absolutely on \(z\in \HH\)

\begin{equation*} \widetilde \theta(z + 2) =\widetilde \theta(z) \end{equation*}
\begin{equation*} \widetilde \theta(-1/z ) = \sqrt{-iz} \widetilde \theta(z) \end{equation*}

in general for \(\gamma \in \Gamma_0(4)\)

\begin{equation*} \widetilde \theta(\gamma z ) = j(\gamma ; z) \widetilde \theta(z) \end{equation*}

where \(j(\gamma; z) = \legendre cd \epsilon_d^{-1} (cz+d)^{1/2}\) where \(\epsilon_d = 1\) if \(d\equiv 1 \pmod 4\text{,}\) -1 \(d\equiv 3 \pmod 4\) sign of Gauss sum. \(\legendre cd\) is a sibling of Legendre form.
